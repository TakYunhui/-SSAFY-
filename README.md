1. 자신의 이름으로 된 폴더 생성
2. {풀이날짜:YYMMDD_문제 출처_문제 번호}

   - github에 풀이 코드 자동으로 올리는 extension 쓰면 제목 안 맞춰도 됩니다.. 그냥 그거 쓰십쇼
   - extension 없이 직접 올리면 되도록 양식 맞춰주길 지향합니다
